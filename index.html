Executive Summary

The Domain Name System (DNS) is like the phonebook of the internet. It helps users find websites using names (like google.com) instead of IP addresses (like 142.250.190.14).

Key Terms

  • DNS: Domain Name System
  • IP Address: A unique string of numbers that identifies a device on the internet
  • Domain: The readable name of a website, like example.com
