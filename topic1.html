What is DNS?

DNS stands for Domain Name System. It helps your computer find websites using names instead of numbers.

Simple Steps

  1. You type a website name like google.com.
  2. Your computer asks a DNS server to find the IP address.
  3. The DNS server sends back the number (IP address).
  4. Your browser uses that number to load the website.
